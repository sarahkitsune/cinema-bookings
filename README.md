# Cinema Bookings

Cinema Bookings is a full-stack web application for managing a cinema booking flow, from movies and rooms to scheduled performances and seat selection. The project is built with React, TypeScript, and Vite on the frontend, with Prisma and PostgreSQL handling data modeling and persistence.

The current domain model includes movies, performances, rooms, seats, bookings, and performance-based seat availability. This makes it possible to track which seats are free, blocked, or booked for a specific show rather than only at room level.

The repository is currently focused on the core booking architecture and database structure, providing a solid base for features such as authentication, checkout, admin management, and real-time booking workflows.
